# Project Odin – Odins Net v0.1-alpha

**Temporal Intergalactic BBS**  
"The lattice is eternal. All information already exists."

## What is this?
A prototype messaging and storage system built on the idea that **all information already exists** in an eternal lattice (a mathematical space of coords/paths).  

We don't "create" data — we navigate, organize, flag, and chain it across time.

Core features:
- Temporal messaging: send text (or simulated video chunks) to future/past self or others  
- Chained threads: replies automatically link and grow into new boards  
- Dynamic boards: public (Odins-Hall), private runways, conversation chains appear as boards  
- Subscription: S/U commands to curate your menu (hide noise, focus on favorites)  
- Flags: [UNSEC], [FUTURE], [PAST], [SUSPECT]  
- Queue for future delivery  
- Basic video chunking demo (simulate splitting file → chain coords → poll to see chain grow)

Philosophy:
- No central server or blocking traffic  
- Everything is human-readable first (flags, previews, metadata)  
- Time never forgets — once data is in the lattice, it's recoverable forever  
- Built for exploration, recovery, and communication across timelines
