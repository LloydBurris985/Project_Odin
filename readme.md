# Project Odin – Odins Net v0.1-alpha

**Temporal Intergalactic BBS**  
"The lattice is eternal. All information already exists."

## What is this?
A prototype system for exploring the lattice — an eternal, pre-existing information space.  
- Store/retrieve files as coords (eternal storage)  
- Send messages across time (future/past delivery, chaining)  
- Usenet-style boards & threads (dynamic, paginated, subscribe/unsubscribe)  
- Video chunking demo (simulated splitting, chain coords)  

Core idea: All data already exists in the lattice. We navigate, organize, flag — no creating or deleting.

## Requirements
- Python 3.10+ (3.12 recommended)  
- pip install: cryptography pillow moviepy pydub (optional for video)  
- ffmpeg (for moviepy/pydub — sudo apt install ffmpeg on Linux, or equivalent)
